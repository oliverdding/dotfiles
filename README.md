# dotfiles

When I'm a student trying to live in terminal, I got tons of dotfiles to manage. At that time I manage the dotfles in one git repository with symbol links.

As time goes by, I got more machine, across different environmen such as MacOS, Linux. To not be stuck at adding if-else on the install-script, I start to use [yadm](https://yadm.io/).

**This repository is all my dotfiles managed by yadm.**

Make yourself at home!
